装饰器decorator

2020-02-08 / 3 min read

装饰器decorator

1 概念理解

  1. Python 中的函数可以像普通变量一样当做参数传递给另外一个函数；
  2. 装饰器本质上是一个 Python 函数或类，它可以让其他函数或类在不需要做任何代码修改的前提下增加额外功能，装饰器的返回值也是一个函数/类对象；
  3. decorator就是一个返回函数的高阶函数；
  4. 装饰器是在原函数外再加一层函数；
  5. 装饰器可以在不修改原函数的情况下在函数前面或后面执行一些代码，或者进行异常捕捉（通过try）。

2 无参数函数装饰器

2.1 实例

import functools

def log(func):
    @functools.wraps(func)
    def wrapper(*args, **kw):
        print('call %s():' % func.__name__)
        return func(*args, **kw)
    return wrapper

@log
def now():
    print('2020-02-08')

now()

执行结果：

call now():
2020-02-08

2.2 知识点

2.2.1 装饰符@

@的作用是将其下的函数（func）作为参数传入@后的函数（test）中，并执行。

实例

def test(f):
    print("before ...")
    f()
    print("after ...")

def func():
    print("func was called")
   
func = test(func)

# 以上代码与下面等效

def test(f):
    print("before ...")
    f()
    print("after ...")

@test  # 相当于执行了func = test(func)
def func():
    print("func was called")

执行结果：

before ...
func was called
after ...

2.2.2 @functools.wraps(func)

经过decorator装饰之后的函数，它们的__name__已经从原来的'now'变成了'wrapper'，被装饰后的函数其实已经是另外一个函数了。为了不影响其属性，使用@functools.wraps(func)将输入函数的属性复制过来。

3 带参数函数装饰器

3.1 实例

import functools

def log(text):
    def decorator(func):
        @functools.wraps(func)
        def wrapper(*args, **kw):
            print('%s %s():' % (text, func.__name__))
            return func(*args, **kw)
        return wrapper
    return decorator

@log('abc')
def now():
    print('2020-02-08')

now()

3.2 知识点

  1. 这里log是一个返回decorator的高阶函数；
  2. @log('abc')的作用是now = log('execute')(now)

4 装饰器进行异常捕捉

4.1 实例1

def d_func(fun):
    def add_cap(*args, **kwargs):
        try:
            fun(*args, **kwargs)
        except:
            print('Error execute: %s' % fun.__name__)
            print(kwargs)
            print(traceback.format_exc())
            sys.exit()
    return add_cap

@d_func
def u_func():  #只在这个函数中用，因此定义在此函数中
    1/0

u_func()  # 调用

4.2 实例2

def run_with_exc(f):
    """游戏运行出现错误时，用messagebox把错误信息显示出来"""
    def call(window, *args, **kwargs):
        try:
            return f(window, *args, **kwargs)
        except Exception:
            exc_info = traceback.format_exc()
            QMessageBox.about(window, '错误信息', exc_info)
    return call
