修改m4a文件详细信息

2020-08-08 Views Python433字2 min read繁

最近喜欢听赵海洋的夜色钢琴曲，想把所有音频下载在电脑上。试了几个客户端，Android和win10的客户端下载都是xm文件，只有win10 UWP应用下载后是m4a文件。但是下载后的m4a文件的详细信息乱码，文件名保存在list.json文件中。所以就有了这篇博客。

一、读取json文件

使用json.load(load_list)方法，将list.json中的json字符串转换成字典。其中"title"是音频名称，"trackId"是下载后的文件名。list.json中每首歌都是一个json字符串，读取进来后是一个列表。

二、修改文件名

使用os.rename(m4aPath, newname)方法。

三、修改文件详细信息

修改详细信息使用了mutagen的mp4模块。修改了详细信息中标题、参与创作的艺术家、唱片集艺术家、唱片集。

四、代码

#-*-coding:utf-8-*-
import json
from mutagen.mp4 import MP4
import os

track_title = '\xa9nam'    # 标题
artist = '\xa9ART'          # 参与创作的艺术家
album_artist = 'aART'       # 唱片集艺术家
album = '\xa9alb'           # 唱片集

rootPath = '291718\\' # m4a文件目录

with open("291718list.json",'r', encoding='UTF-8') as load_list:
    load_dicts = json.load(load_list)
    for load_dict in load_dicts:
        # 获取文件名
        title = load_dict["title"].replace('夜色钢琴曲', '')
        title = title.replace(' - 赵海洋', '')
        title = title.replace('赵海洋', '')
        title = title.replace('-', '')
        title = title.replace('《', '')
        title = title.replace('》', '')
        title = title.strip()
        # 修改详细信息
        m4aPath = rootPath + str(load_dict["trackId"]) + ".m4a"
        songFile = MP4(m4aPath)
        songFile[track_title] = title
        songFile[artist] = "赵海洋"
        songFile[album_artist] = "赵海洋"
        songFile[album] = "夜色钢琴曲"
        songFile.save()
        # 修改文件名
        newname = rootPath + "《夜色钢琴曲》" + title + " - 赵海洋" + ".m4a"
        os.rename(m4aPath, newname)
        print(newname)

print("Done!")
