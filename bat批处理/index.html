以上语句中 3 既是延时三秒。

其他

1. 注释

两个冒号 ::

参考资料：bat批处理的注释语句

2. 重启资源管理器

taskkill /IM explorer.exe /F
start "" explorer.exe

3. cmd中文乱码（修改编码）

定位到注册表 HKEY_CURRENT_USER\Console%SystemRoot%_system32_cmd.exe

新建一个 DWORD（32位值）,命名为CodePage，值设为65001

已有CodePage的话，修改它，改为十进制，65001

代码页 描述
65001 UTF-8
950 繁体中文
936 简体中文默认的GBK
437 MS-DOS 美国英语
